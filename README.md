# Quantity Measurement Spring Boot Application

A Spring Boot application providing APIs for converting and managing various physical quantities like Length, Volume, Weight, and Temperature.

## Features
- **Conversion APIs:** Perform conversions between different units of measurement (e.g., Feet to Inches, Liters to Gallons).
- **RESTful Architecture:** Uses standard HTTP methods for interactions.
- **Robust Error Handling:** Global error and exception scenarios handled gracefully.

## Tech Stack
- Java
- Spring Boot
- Maven
